Privacy Enhancements

As a background process, you can reset all browser data (except passwords).

(KEEP IN MIND THIS CLEARS ALL BROWSER DATA. PASSWORDS ARE NOT AFFECTED)
This feature will clear:

  • Cache
  • Bookmarks (Not cleared due to a bug)
  • Cookies
  • Download History
  • Form data
  • Browsing history
  • Indexed Databases (storage)
  • Local Storage
  • Plugin Data
  • Passwords
  • Server Bound Certificates
  • Service Workers
  • Last opened tab
  • window.name object
